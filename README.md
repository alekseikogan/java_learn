# Введение в Front-End разработку (Meta)

Этот репозиторий содержит проекты и практические задания, выполненные в рамках курса **"Введение в Front-End разработку"** от Meta.

## Содержимое

- Основы HTML и CSS  
- Работа с семантическими тегами и структурой страницы  
- Формы, таблицы и элементы управления  
- Применение базовых JavaScript-концепций для интерактивности  
- Практические упражнения и мини-проекты  

## Цель

Цель репозитория — систематизация изученного материала и демонстрация навыков Front-End разработки, полученных на курсе.
